# 数字图像处理与机器学习代码库

本仓库包含数字图像处理和机器学习相关的代码实现，适合入门学习参考。
图像处理参考冈萨雷斯第四版，机器学习参考西瓜书。

## 仓库结构

- `digital_image_process/`：数字图像处理算法（滤波、边缘检测等）
- `machine_learning/`：机器学习基础模型（分类、回归等）
- `img/`：项目所需图片资源（还未更新）

## 环境要求

- Python 3.7+
- 依赖库：numpy、opencv-python、scikit-learn、matplotlib

## 使用方法

1. 克隆仓库：
